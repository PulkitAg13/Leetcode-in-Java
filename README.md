## 🛠️ How to Run the Code

1. Clone the repository:

   ```bash
   git clone https://github.com/PulkitAg13/LeetCode-in-Java.git
   ```
2. Navigate to the solutions folder:

   ```bash
   cd LeetCode-in-Java/solutions
   ```
3. Compile and run any file:

   ```bash
   javac Leetcode1.java
   java Leetcode1
   ```
